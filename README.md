# Wedding Journey - February 2026

Our wedding trip itinerary from Korea to Germany and Copenhagen.

## Preview

A romantic timeline-style schedule page showing our journey.

## How to Run

### Option 1: PHP Server
```bash
php -S localhost:8000
```
Then open http://localhost:8000 in your browser.

### Option 2: XAMPP/MAMP
Place the folder in your `htdocs` directory and access via localhost.
